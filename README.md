# E-Commerce-Sales-Analysis
Used Global E-Commerce Sales and Customer Data dataset from kaggle
An end-to-end data analytics project focused on analyzing sales, profit, shipping cost, and customer behavior using SQL for data preparation and Power BI for interactive dashboards. The project provides insights into regional performance, product profitability, payment trends, and shipping impact on business outcomes.
Tools Used 
• SQL (Data cleaning & analysis) 
• Power BI (Dashboard creation & visualization) 
Key KPIs 
• Total Sales 
• Total Profit 
• Total Orders 
• Average Order Value
